# IMDB Reviews

### How to run the Code

1. Move the IMDB reviews data file, `aclImdb`, to this file.

2. Open jupyter notebook under the file

   ```
   $ jupyter notebook
   ```

3. Select  `IMDB_review.ipynb` and run it .
